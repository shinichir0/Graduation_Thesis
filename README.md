# Graduation_Thesis
## document
各章ごとのtexファイル
## draw.io
draw.ioで作成した図のxmlファイル
draw.ioとgithub apiで連結
## fig
各図ファイル
## material
関連研究やその他のpdfファイル
## reference
bibtex用のbibファイル
## stylefile
各スタイルファイル
### 現在使用しているもの
- bpaper.sty(\\thesisなどの設定)
- fancyheading.sty(chapterの書式設定)
- jtygm.sty(linuxでtexliveの際のフォント変換設定(もともとerrorではなくwarningなのでなくてもよい))
